Introduction

This is a small application to manage ships that will allow the user to perform CRUD (Create, Read, Update & Delete) operations on a ship. Each ship must have a name (string), length (in meters), width (in meters), and code (a string with a format of AAAA-1111-A1 where A is any character from the Latin alphabet and 1 is a number from 0 to 9).

The project has been created using the .Net Core Web API (Dependency Injection — Auto Mapper — Repository Pattern – Entity Framework In-Memory), Angular 10+ as frontend, and Docker as a deployment platform.
