Introduction

This is a small application to manage bookings for trains, buses, and planes. It will allow the user to Register, Login, Search Bookings, Book Ticket,
Generate Confirm Ticket XML, View Ticket History, View All Users Bookings, and Download User Bookings.

The project has been created using the .Net Core Web API (Dependency Injection — Repository Pattern – Entity Framework In-Memory), Angular 16+ as frontend.
You can kick start and test the application by already added data in the in-memory database.
When application is run it will create three users into the database.
